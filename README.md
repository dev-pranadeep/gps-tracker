# GPS Tracker

A full-stack GPS Tracking System built with React, Node.js, Express, and MongoDB.

## Features

- User Registration & Login
- JWT Authentication
- Forgot Password (Email)
- Admin & User Roles
- Live GPS Tracking
- Interactive Leaflet Map
- Search Users
- Online / Offline Status
- Google Maps Navigation
- Distance Between Admin and User

## Technologies Used

Frontend
- React
- React Router
- Axios
- React Leaflet
- Leaflet

Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT
- Bcrypt
- Nodemailer

## Installation

Clone the repository

```bash
git clone https://github.com/dev-pranadeep/gps-tracker.git
```

Install dependencies

```bash
npm install
```

Create a `.env` file using `.env.example`.

Run the project

```bash
npm run dev
```
